- Group Supported

## Environment Variables

To run this project, you will need to add the following environment variables 

`API_ID`

`API_HASH`

`BOT_TOKEN`


## Deployment

To deploy this project run

```bash
  pip install -r requirements.txt
  python3 main.py
```
